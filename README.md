# 💻 &nbsp; PPC Scripts

This repo includes some scripts to query PPC (Prize Pool Contributions) at specific timestamps.

## 💾 &nbsp; Installation

Make sure you have [pnpm](https://pnpm.io/) installed, as it is the package manager used for this repo.

`pnpm i`

## 🏎️ &nbsp; Quickstart

To run queries, use `pnpm start`.
